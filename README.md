# Pokemon Battles

## Start Backend

```
cd backend
npm i
npm start
```

## Start Frontend

```
cd frontend
npm i
npm run dev
```

## Environment

- `VITE_BACKEND_URL` (optional, defaults to `http://localhost:3000`)

## Replay

- Use the Replay tab to load a Replay ID
- Or call `GET /replays/:id` directly on the backend
